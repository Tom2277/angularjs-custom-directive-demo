This is a less maintained demo-page demonstrating that you can hard-load a full HTML page with boilerplate within the same AngularJS app using the ui-router elsewhere.

You might want to force page refreshes on content changes to force a refresh of ads views on new content. Thankfully the days of those needed work-arounds are coming to an end.

Note.. this will break some page links in your shared navbar. You'll need a second navbar template without 'ui-sref' attributes for statically loaded pages.
